### Material Properties

| Material | Property Values |
| :--- | :--- |
| **Water** | $C_p = 4.187 \frac{\text{kJ}}{\text{kg}\cdot K}$ <br> $C_p = 1.0 \frac{\text{BTU}}{\text{lb}\cdot R}$ |
| **Steam** | $C_p = 2.0 \frac{\text{kJ}}{\text{kg}\cdot K}$ <br> $C_v = 1.5 \frac{\text{kJ}}{\text{kg}\cdot K}$ <br> $L_v = 2257 \frac{\text{kJ}}{\text{kg}}$ <br> $k = 1.33$ |
| **Ice** | $C_p = 2.09 \frac{\text{kJ}}{\text{kg}\cdot K}$ <br> $C_p = 0.5 \frac{\text{BTU}}{\text{lb}\cdot R}$ <br> $L_f = 335 \frac{\text{kJ}}{\text{kg}}$ <br> $L_f = 144 \frac{\text{BTU}}{\text{lb}}$ |
| **Air** | $C_p = 0.24 \frac{\text{BTU}}{\text{lb}\cdot R}$ <br> $C_v = 0.17 \frac{\text{BTU}}{\text{lb}\cdot R}$ <br> $R = 0.287 \frac{\text{kJ}}{\text{kg}\cdot K}$ <br> $R = 53.34 \frac{\text{ft}\cdot\text{lb}}{\text{lb}\cdot R}$ <br> $k = 1.4 \text{ (cold)}, 1.3 \text{ (hot)}$ |
| **Density** | $\rho_{\text{air}} = 1.205 \frac{\text{kg}}{\text{m}^3} \text{ at } 20^\circ\text{C}$ <br> $\rho_{\text{water}} = 1000 \frac{\text{kg}}{\text{m}^3} \text{ at } 20^\circ\text{C}$ |
